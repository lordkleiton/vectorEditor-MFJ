Instruções

Clique na tela para selecionar o ponto inicial do vetor

Mova o mouse dentro da área para definir o tamanho do vetor

Clique no ponto que desejar para encerrar o vetor atual

Pressione Esc para parar de criar novos vetores

Caso tenha parado de criar novos vetores e deseje voltar a criar, basta clicar novamente na área de desenho

Pressione limpar para recomeçar

Pressione shuffle para aleatorizar os vetores
